🏴‍☠️PIRÁTSKE STRÁNKY

Čo sú pirátske stránky?

Pirátske stránky sú webové stránky, ktoré ponúkajú nelegálne šírený obsah, ako sú:

  • Filmy a seriály (napr. torrentové stránky, streamovacie servery)
  • Hry, softvér (napr. cracknuté verzie programov)
  • Hudba, knihy a iné médiá

Známe pirátske stránky

  • The Pirate Bay
  • 1337x
  • RARBG (už nefunkčný)
  • Fmovies
  • ZLibrary

Ako fungujú?

Zvyčajne ide o weby, ktoré umožňujú používateľom sťahovať alebo streamovať obsah cez torrenty alebo priame odkazy.
Prevádzkovatelia týchto stránok často zarábajú na reklamách alebo daroch.

Aké sú riziká pri používaní?

  • NELEGÁLNOSŤ: sťahovanie chráneného obsahu je vo väčšine krajín nelegálne
  • MALVÉR: tieto stránky môžu obsahovať škodlivý softvér
  • SLEDOVANIE: mnohé krajiny monitorujú IP adresy používateľov, ktorí sťahujú obsah cez torrenty

🌑DARK WEB

Čo je to dark web?

Dark web je časť internetu, ktorá nie je indexovaná bežnými vyhľadávačmi (ako Google) a je prístupná len pomocou špeciálneho softvéru, ako je Tor browser.

Rozdelenie internetu

  • Surface web – bežné stránky, ktoré nájdete cez Google (4-5% internetu)
  • Deep web – stránky, ktoré nie sú verejne prístupné (90% internetu napr. súkromné databázy, e-mailové účty)
  • Dark web – anonymný internet, prístupný len cez šifrované siete ako Tor

Deep web vs Dark web

Vlastnosť Deep web Dark web
Prístupnosť Vyžaduje prihlasovanie (napr.heslo) Vyžaduje špeciálny softvér (Tor)
Legálnosť Legálny (napr. súkromné účty) Môže byť aj legálny, ale často nie
Vyhľadávanie Nie je indexovaný v Google Nie je indexovaný v Google
Obsah Údaje, ktoré nechceme verejne zdieľať Anonymné trhy, fóra whistleblowing
Príklady Gmail, Netflix účet, vedecké archívy Silk Road, AlphaBay, SecureDrop

Čo nájdeme na Dark webe?

  • Legálne veci: fóra, whistleblowing platformy (napr. SecureDrop, ProPublica)
  • Nelegálne veci: trhy s drogami, zbraňami, falošnými dokladmi, hackerské služby, úniky dát

Známe "temné" trhoviská

  • Silk Road (zrušený)
  • AlphaBay
  • Hydra

Aké sú riziká pri používaní?

  • Podvody: veľa ponúk je falošných, podvodných
  • Právne riziká: prístup a nákup nelegálneho tovaru môže viesť k trestnému stíhaniu
  • Malvér a sledovanie: aj cez Tor hrozí napadnutie zariadenia alebo sledovanie (napr. štátnymi agentúrami)

UPOZORNENIE

Pirátske stránky a dark web môžu ponúkať veci, ktoré sa javia ako „zadarmo“ alebo „anonymné“, no väčšinou ide o rizikové a nelegálne aktivity.
Ak chceš bezpečne a legálne používať internet,vyhýbaj sa týmto stránkam a službám.
